# PiWork Product Notes

PiWork is for everyday work that starts with context and ends with a deliverable.

Core loop:

1. Index one or more folders you already trust.
2. Reference them in chat with `@alias`.
3. Let PiWork generate notes, briefs, recaps, and drafts into `~/piwork`.

Important constraints:

- indexed folders are read-only context
- generated work lives in PiWork-owned storage
- chat should stay concise; longer output belongs in artifacts
